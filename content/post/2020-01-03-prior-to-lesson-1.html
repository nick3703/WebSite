Welcome to MA376! Prior to class please ensure you have a version of R Available Here and R Studio, Available Here. THe majority of this course will be conducted in R, but if you want to use Python, there’s nothing stopping you!

In R Studio run library(tidyverse) If you get an error, you need to install the package via install.packages("tidyverse"). If you need a referesher on tidyverse, I’d highly recommend R4DS.

The textbook for this course is Intermediate Statistical Investigations which can be purchased through the bookstore. We will also be doing explorations from the textbook which involve the data sets located at http://www.isi-stats.com/isi2/applets.html.

Please read the course website and the course calendar at https://www.nicholasjsclark.com/courses/testpost/ or through clicking the link at the top of this page.

In this course we will be working through a variety of statistical explorations. My expectation is that you complete the assignment using Rmarkdown preferably knit to pdf. If you have issues knitting to pdf the first thing to do is ensure you have MikTex installed. One resource for setting this up is at This Website. If you continue to have problems knitting to pdf you may have to follow the follwing steps (credit to Timesaver from this issue:

  1. Start MikTex console
  2. Click on “Restart as Adminstrator”
  3. Then choose “Always install missing packages on the fly”
  4. Close the console
  5. Restart RStudio
