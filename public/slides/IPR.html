IPR for ERCA-II

Bayesian FMEA

  • If a component of a system goes wrong then two things must have happened, component failed and we did not detect it/mitigate it

Definitions

  • \(Z_j = 1\) if component \(j\) fails, \(0\) otherwise
  • \(p_j=\) Probability of Component \(j\) failing
  • \(d_j|Z_j=1\) Probability of detecting component given that it has failed
  • \(F_j\) is failure time

Occurrence Probabilities

  • Prob Occurrence = 3
  • Unlikely to occur some time in the life of an item; 10-49%
  • First approach:
  • Assumption, there is no aging. What we are saying is that there is a 10-49% chance of a failure occurring in L trials where L is lifetime
  • \(F_j \sim Geom(p_j)\)
  • \(P(F_j\leq L)= (1-(1-p_j)^L)\)
  • e.g. if \(L=10\), \(p_j \in (.01,.06)\)
  • \(Z_j \sim Ber(p_j)\)

To play around with

p.func<-function(L){
  p.low<- 1-(1-.1)^(1/L)
  p.high<- 1-(1-.49)^(1/L)
  return(list(p.low=p.low,p.high=p.high))
}

Occurrence Probabilities

estBetaParams <- function(mu, var) {
  alpha <- ((1 - mu) / var - 1 / mu) * mu ^ 2
  beta <- alpha * (1 / mu - 1)
  return(params = list(alpha = alpha, beta = beta))
}
values<-p.func(10)
O3.parameters<-estBetaParams((values$p.low+values$p.high)/2,.01^2)

Occurrence Probabilities

sim.dat<-data.frame(x=rbeta(10000,O3.parameters$alpha,O3.parameters$beta))
sim.dat%>%ggplot(aes(x=x))+geom_density()

Occurrence Probabilities

-Benefit of this methodology is updating \(p_j\) is straightforward, given a test of size \(n\) with \(f\) failures:

\(f(p_j|Test)= Beta(\alpha^*=\alpha+f,\beta^*=\beta+n-f)\)

Alternatively

  • Model as lifetime, failure rate is constant 10%-49%
  • Lifetime of component \(j\) modeled as \(L_j \sim Exp(\lambda_j)\) where \(\lambda_j\) is risk function
  • \(1-\exp(-\lambda_j L_j)\in(.1,.49)\)
  • \(\lambda_j \in (-\frac{\log(.9)}{L_j},-\frac{\log(.1)}{L_j})\)

Prior for \(\lambda_j\)

  • For instance if \(L_j=\) 1 year, we can set prior using conjugate Gamma, since \(\lambda \in (.1,2.3)\)
estGammaParams <- function(mu, var) {
  alpha <- mu^2/var^2
  beta <- var^2/mu
  return(params = list(alpha = alpha, beta = beta))
}
lam.prior=estGammaParams(1.2,1)

Prior for \(\lambda_j\)

sim.dat<-data.frame(x=rgamma(10000,lam.prior$alpha,scale=lam.prior$beta))
sim.dat%>%ggplot(aes(x=x))+geom_density()

Posterior after testing

  • A bit tricky since data is potentially right censored
  • \(p(\lambda|Test)\propto \prod_{fails}f(t|\lambda)\prod_{cens}(1-F(t|\lambda))p(\lambda)\)
  • Probability of component failing then depends on how long it is observed
